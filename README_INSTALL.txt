INSTALLATION INSTRUCTIONS FOR SUMMER BREEZE WEB PANEL

1. Unzip this folder.
2. Go to https://github.com and open your repository:
   https://github.com/summerbrz/SummerBreezeAI.github.io

3. Tap "Add file" → "Upload files"
4. Upload:
   - index.html
   - panel.js
   - listener.js

5. Scroll down and click "Commit changes"

6. Go to Settings → Pages
   - Set source to: "Deploy from branch"
   - Branch: main, Folder: / (root)
   - Click Save

7. After ~30 seconds, your site is live at:
   https://summerbrz.github.io/SummerBreezeAI.github.io/

8. To add a backend (like MindPal, n8n, or Ollama), update the iframe src in index.html

You're now running a frontend-ready AI interface for Summer Breeze!
